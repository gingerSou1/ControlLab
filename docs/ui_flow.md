# UI Flow

The Control Head UI is page-based and navigated via keypad inputs.

---

## Pages

- STATUS – System overview
- CONTROL – Operator setpoint and commands
- ALERTS – CAS display and acknowledgment
- MAINT – Diagnostics and configuration

---

## Keypad Mapping

| Key | Function |
|---|---------|
| A | STATUS |
| B | CONTROL |
| C | ALERTS |
| D | MAINT |
| * | ACK / Back |
| # | Enter |

---

## Mode Restrictions

- CONTROL page disabled in NORMAL without enable
- MAINT page requires unlock
- Setpoints ignored in NORMAL unless within limits
